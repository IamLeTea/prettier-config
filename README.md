# `@letea/prettier-config`

> My [Prettier](https://prettier.io) config.

## Usage

**Install**:

```bash
$ yarn add --dev @letea/prettier-config
```

**Edit `package.json`**:

```jsonc
{
  // ...
  "prettier": "@letea/prettier-config"
}
```
